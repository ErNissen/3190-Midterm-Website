We retrieved current weather data using the free public api at openweathermap.com

For the sake of practicing populating an HTML page using JavaScript and JSON, the results of the 5-Day Forecast page are not real. We apologize for the inconvience.

You can view the authors of this website by clicking "authors" in the footer of this webpage, or by clicking here.

In the future, we plan on adding a fully capable weather forecast, including hourly weather, true 5-day weather, and live public weather feed. Unfortunately, a lot of APIs for more detailed weather information follow paid models

There are a few things I would recommend. I personally add my class schedule, birthdays, appointments, and sometimes important reminders.
